# Flutter Map Markers Example

This example app demonstrates the capabilities of the `flutter_map_markers` package through various interactive demo pages. The app showcases different types of markers, animations, and utility functions that can be used with Flutter Map.

## Demo Pages

The app includes the following demonstration pages accessible through the navigation drawer:

###  **Marker Demo**
Displays a large number of randomly generated markers (2000+) clustered around London with various icons and colors. Demonstrates performance with many markers and includes a slider to adjust the marker count dynamically.

###  **Simple Marker**
Basic marker functionality with tap-to-add markers. Shows how to create simple raindrop-style markers and handle tap events with coordinate display in snackbars.

###  **Price Tag Marker**  
Demonstrates text-based markers displaying price information (£500-£5000) scattered around London. Shows how to create custom text markers with background styling.

###  **Animated Marker**
Interactive marker animation demo where tapping the map adds new markers with smooth size animations. Demonstrates how to implement custom marker animations using Flutter's animation framework.

###  **Meters to Pixels**
Visualizes how to convert real-world distances (meters) to screen pixels at different zoom levels. Shows a 10km radius circle that scales appropriately with map zoom.

###  **LatLng to Offset** 
Demonstrates coordinate conversion by drawing connected lines between markers in a random walk pattern. Shows how to convert latitude/longitude coordinates to screen offsets for custom drawing.

###  **Ball Marker**
Simple circular ball-style markers with custom paint styling including gradients and stroke effects.

## Getting Started

Run the app and use the drawer menu (hamburger icon) to navigate between different demo pages. Each demo showcases specific features of the flutter_map_markers package and serves as a reference for implementation.
